# learning-log

Практическая лаборатория для изучения сетей, системного администрирования и автоматизации. Этот репозиторий служит подтверждением моих компетенций (Proof of Work) и фиксирует реальный практический опыт.

## 🛠 Технологический стек и направления
- **Сети:** Модель OSI, стек TCP/IP, анализ трафика и пакетов (Wireshark)
- **Автоматизация:** Разработка скриптов на Python (с соблюдением стандарта PEP 8)
- **Безопасность ОС:** Администрирование Linux, bash-скрипты, настройка прав доступа

## 📁 Структура репозитория
- `/networking` — Дампы сетевого трафика (.pcap), схемы сетевой архитектуры и разборы протоколов.
- `/automation` — Скрипты и мини-утилиты на Python для сетевых задач и автоматизации безопасности.
- `/linux` — Конфигурационные файлы, скрипты автоматизации для Linux и логи настройки системы.
